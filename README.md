# nodejs-bitcoin
A simple Mercado Bitcoin trader bot.

## Instructions

1. Run npm install to download the dependencies
2. Create a .env root file with your configs (see the template at .env.example)
3. Run npm start to start monitoring Mercado Bitcoin (it takes some seconds to start)
