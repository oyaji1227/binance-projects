## Instruções

1. Execute npm install na pasta do projeto para baixar dependências
2. Obtenha suas credenciais de API do Mercado Bitcoin em https://www.mercadobitcoin.com.br/trade-api/
3. Crie um arquivo .env na raiz do projeto com o mesmo conteúdo do arquivo .env.example
4. Preencha os valores das variáveis no .env conforme instruído nos comentários (#)
5. Ajuste a linha 32 do index.js para sua lógica de 'comprar barato'
5. Execute o robô com npm start, leva alguns segundos para começar (CRAWLER_INTERVAL)
